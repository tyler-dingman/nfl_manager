# Mobile QA Checklist

Use this list when validating the Falco mobile experience.

## Layout + Overflow
- No horizontal scroll on any screen.
- Tables or long lists stay within the viewport.
- Sticky headers/footers do not overlap core actions.

## Navigation
- Team select → Experience → Roster/Free Agency/Draft works.
- Back navigation returns to expected screen.
- Nav items remain tappable and visible.

## Modals
- Modal content scrolls when tall.
- Close button visible and tappable.
- Background scroll is locked when modal open.
- No content hidden behind keyboard.

## Forms
- Inputs focus correctly and accept typing.
- Submit buttons remain visible and tappable.
- Validation messages fit within modal.

## CTAs
- Primary actions are visible without horizontal scroll.
- Buttons are not blocked by overlays/ads/sticky elements.

## Draft
- Draft board and available players are readable.
- Action buttons are visible and tappable.

## Trade
- Offer pickers scroll safely.
- Action buttons remain visible.
